# 免费官方案例来源登记表

## 来源层级

| 等级 | 来源 | 可支持的结论 | 主要限制 |
|---|---|---|---|
| A1 | 最高人民法院指导性案例、人民法院案例库入库案例的官方全文页 | 案例身份、裁判要点、已公开事实与结果 | 仍需核验后续法源变化 |
| A2 | 最高人民法院公报案例、官方完整裁判文书 | 已公开裁判事实、理由和结果 | 检查审级及是否有后续裁判 |
| B | 最高人民法院或法院官方典型案例摘要 | 官方概括的规则、事实和结果 | 常匿名并缺案号、法院或完整理由 |
| C | 法院官方新闻、案例解读、答记者问 | 发现线索、理解官方说明 | 不是裁判文书，不单独支撑具体裁判事实 |
| D | 搜索摘要、媒体、律所文章、商业数据库摘要 | 发现线索 | `lead-only`，不得作正式案例依据 |

## 首选入口

1. 人民法院案例库：`https://rmfyalk.court.gov.cn/`
   - 用于指导性案例和入库参考案例。
   - 公开主页可访问；检索或详情可能要求登录。不得调用未授权接口或绕过登录。
2. 最高人民法院：`https://www.court.gov.cn/`
   - 搜索指导性案例、典型案例、入库案例转载、司法政策和发布说明。
   - 常用检索式：`site:court.gov.cn 争点 指导案例`、`site:court.gov.cn 争点 入库参考案例`、`site:court.gov.cn 争点 典型案例 裁判结果`。
3. 最高人民法院公报：`https://gongbao.court.gov.cn/`
   - 用于公报案例及最高法规范性发布。
4. 中国裁判文书网：`https://wenshu.court.gov.cn/`
   - 用于普通裁判文书；可能有登录、验证码、脚本或维护限制。
5. 专业法院及法院官方站点：`*.court.gov.cn`、`*.chinacourt.gov.cn`
   - 用于知识产权、海事、金融及地方法院案例；核对网站主办机关和裁判文书身份。
6. 多元解纷案例库：`https://dyjfalk.court.gov.cn/`
   - 仅用于调解、仲裁等非诉解纷案例；不得与法院生效裁判混同。

## 查询阶梯

对每个争点至少建立三组查询：

1. **核心查询**：案由/法律关系 + 核心争点。
2. **事实查询**：核心查询 + 决定性事实或行为模式。
3. **反向查询**：核心查询 + `不予支持`、`驳回`、`不构成`、`无效`、`未证明` 等反向结果词。

再按来源类型追加：`指导案例`、`入库参考案例`、`典型案例`、`公报案例`、`民事判决`、`二审`、`再审`。

## 官方性判断

- URL 属于官方域名只是起点；还要核对页面标注的来源、发布机关和内容类型。
- `court.gov.cn` 上的人民法院报转载可作为官方公开材料，但若缺案号或法院，按 B/C 级处理。
- PDF 或附件必须回到其官方落地页；只有搜索引擎缓存链接时不得标记 `verified-source`。
- 同一案例多个官方页面不一致时，保留全部链接并报告冲突。
