# Pleading quality gate

Pass only when each material fact and requested relief/defence is supported by supplied material, legal propositions are tied to current verified authority, and the document has passed structured checks plus risk-proportionate native preview/native-application inspection. Fail or hold for review when identity, amount, date, forum, case number, procedural posture, core evidence, Chinese fonts, pagination, tables or print layout conflict. Keep drafting, internal review, sending, filing, and signing as separate authorization states.
