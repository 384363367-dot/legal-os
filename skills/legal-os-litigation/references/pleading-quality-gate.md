# Pleading quality gate

Pass only when each material fact and requested relief/defence is supported by supplied material, legal propositions are tied to current verified authority, and the document has passed structured checks plus risk-proportionate native preview/native-application inspection. A complaint, application or answer also requires a separately generated evidence catalogue whose numbering, evidence names and proof purposes match the pleading's factual structure; the pleading itself must not contain an independent “证据和证据来源” chapter. If evidence is incomplete, label the catalogue `待补证 / 内部草稿` and keep the release state blocked or draft-only. Fail or hold for review when identity, amount, date, forum, case number, procedural posture, core evidence, cross-artifact mapping, Chinese fonts, pagination, tables or print layout conflict. Labour/personnel arbitration pleadings fail if they request arbitration costs from the other party. Keep drafting, internal review, sending, filing, and signing as separate authorization states.
